### How to use on Local Machine:

> 1. $  git clone https://github.com/PsKalsi19/Book-Freaks.git
> 2. $  cd Book-Freaks
> 3. $  npm install

> ### Features
* Home Page
* Product Listing Page
  * Add to / Remove from wishlist
  * Show Product Overview Page
  * Add to Cart
* Single Product Page
  * Add to / Remove from wishlist
  * Add to Cart
* Filters by
  * Price
  * Category
  * Ratings
* Sort by
  * Price: High to Low
  * Price: Low to High
* Wishlist Management
  * Add Item
  * Remove Item
  * Move to Cart
* Cart Management
  * Increase / Decrease Quantity
  * Remove Item
  * Move to Wishlist
* Search
  * Item overview on click
* Loading & Alerts
* Authentication
  * Sign-up
  * Login
  * Logout
* Address Management
  * Add / Remove / Set Primary
* Checkout
  * Apply Coupon
* Payment Gateway
* Fully Responsive
* 404 Page
* Experimental: Wishlist and Cart items persist after reload for only existing user and not for the users created through sign up.

> ### Built using
* React
* React Router
* Tailwind
* Mockbee
* Headless UI
* React Hot Toast
* Razorpay Payments 
* React Icons
